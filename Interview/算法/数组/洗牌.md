# 洗牌算法

## 全局算法

C++17支持`random_shuffle()`算法，可以直接就搞

```C++
    void RandomCard_Gloable(){
        vector<int> tem;
        srand(0);
        int length=tem.size();
        for(int i=0;i<length;i++){
            int value=rand()%length;
            swap(tem[i],tem[value]);
        }
    }
```
